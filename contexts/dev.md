# 開発コンテキスト (Development Context)

Mode: Active development
Focus: Implementation, coding, building features

## 振る舞い (Behavior)
- 説明の前にコードを書く
- 完璧な解より動く解を優先する
- 変更後にテストを実行する
- コミットはアトミックに保つ

## 優先順位 (Priorities)
1. 動かす
2. 正しくする
3. きれいにする

## 優先して使うツール (Tools to favor)
- コード変更には Edit、Write
- テスト/ビルド実行には Bash
- コード検索には Grep、Glob
