## ポリシー (Policy)

1. ライセンスのない、または互換性のないソースからの直接コードコピーは行いません。
2. ECC の実装は、このリポジトリのアーキテクチャとライセンスモデル向けに再執筆されています。
3. 参照資料は、ライセンスが明示的に再利用を許可しない限り、アイデア、パターン、概念的フレーミングのみに使用します。
4. 将来の直接再利用には、明示的なライセンス検証と、ファイル内およびリリースノートでのソース帰属が必要です。
