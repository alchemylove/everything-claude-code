# Everything Claude Code リサーチプレイブック (Everything Claude Code Research Playbook)

ドキュメントが多いタスク、ソースが重要なタスク、リポジトリ全体のコンテキストが必要なタスクで使う。

## デフォルト (Defaults)

- 一次ドキュメントと直接のソースリンクを優先する。
- 事実が時間とともに変わりうる場合は具体的な日付を含める。
- 各推奨事項や結論ごとに短い証拠の痕跡を残す。

## 推奨フロー (Suggested Flow)

1. まずローカルのコードとドキュメントを調べる。
2. 不安定な事実や外部の事実だけをブラウズする。
3. ファイルパス、コマンド、リンク付きで調査結果を要約する。
